| Function         | Keybind                                              |
|------------------|------------------------------------------------------|
| Move Camera      | Press Mouse-Wheel Button and Move your Mouse         |
| Zoom In/Out      | Scroll with Mouse-Wheel                              |
| Draw             | Hold Left Mouse Button                               |
| Undo Last-Stroke | CTRL + Z                                             |
